Fitness Center Management & Appointment System

This project is a web-based Fitness Center Management and Appointment System built with ASP.NET Core MVC.
It aims to apply the concepts learned in the Web Programming course by developing a real-world solution for managing gym services, trainers, members, and AI-supported exercise recommendations.

Features

Gym Management: Define fitness centers, working hours, service types, durations, and prices.

Trainer Management: Add trainers, manage their expertise areas, available hours, and supported service types.

Member & Appointment System: Members can book appointments based on trainer availability and service details.

Appointment Validation: Prevents overlapping or invalid bookings and includes an approval mechanism.

Reporting & REST API: Provides data reporting using RESTful APIs.

Technologies Used

ASP.NET Core MVC

Entity Framework Core

SQL Server

REST APIs

AI-based recommendation logic (optional)

Purpose

The project demonstrates the integration of backend development, database design, user interaction, and modern web technologies to solve a real problem in gym/fitness center operations.
